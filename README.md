# KeePassXC

[OpenBSD](https://www.openbsd.org) port of `keepassxc`.

For more information please visit [keepassxc's repo](https://github.com/keepassxreboot/keepassxc).

## Installation

```shell
$ doas pkg_add keepassxc
```
